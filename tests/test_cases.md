## TC-03：工具栏操作

### TC-03.1 加急/取消加急
```
1. 选中行（序号=7）
2. 点击加急按钮 @(700, 118)
3. → 弹出消息提示"操作成功"（消息类型，约100毫秒后渲染）
4. 点击取消加急按钮 @(778, 118)
5. → 弹出消息提示"操作成功"
```

**验证点：**
- [ ] 加急前置条件：行必须已选中，否则弹出通知提示"请先勾选数据"
- [ ] 加急成功出现消息提示"操作成功"
- [ ] 取消加急同样出现"操作成功"
- [ ] 消息提示约 2-3 秒后自动消失

### TC-03.2 批量编辑优先级
```
1. 选中行
2. 点击批量编辑优先级 @(605, 118)
3. → 弹出窗"批量编辑优先级"
    字段: 优先级 (数字输入框, 初始值=0)
    按钮: 增加 / 减少 / 取消 / 确定
4a. 点击增加按钮 → 值: 0→1
4b. 点击输入框聚焦 → 键入(5) → 值: 5
5. 点击确定 → 消息提示"操作成功", 弹窗关闭
```

**验证点：**
- [ ] 弹窗包含数字类型输入框
- [ ] 增加/减少按钮每次变化 ±1
- [ ] 直接输入可覆盖值
- [ ] 确定后弹窗关闭 + 消息提示反馈
- [ ] 取消关闭弹窗不保存

## TC-06：坐标系统验证

| 场景 | 坐标系 | 预期 |
|------|--------|------|
| `获取元素坐标(路径表达式)` → `坐标点击()` | 顶层视口 | ✅ 命中 |
| `子页面.鼠标链移动到(x,y)` | 子页面相对 | ✅ 命中 |
| `页签.鼠标链移动到(x,y)` | 顶层视口 | ✅ 配合视口中心点 |
| 裸用`获取边界矩形()`不叠加偏移 | 子页面相对 | ❌ 点偏 |

**统一工具函数：**
- `获取元素坐标(路径表达式, 索引, 超时)` → 返回 `{水平坐标, 垂直坐标}`
- `获取元素中心点(元素对象)` → 返回 `{水平坐标, 垂直坐标}`（基于视口中心点）
